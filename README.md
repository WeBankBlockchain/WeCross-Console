# WeCross-Console

WeCross-Console是[WeCross](https://github.com/WeBankFinTech/WeCross)的重要交互式客户端工具。

## 关键特性

- 命令支持脚本格式
- 提供了区块链状态的查询命令
- 提供了针对跨链资源的操作命令


## 源码编译

**环境要求**:

  - [JDK8及以上](https://fisco-bcos-documentation.readthedocs.io/zh_CN/latest/docs/sdk/sdk.html#id1)
  - Gradle 5.0及以上

**编译运行如下命令**:

```shell
$ cd WeCross-Console
$ ./gradlew build
```
如果编译成功，将在当前目录生成一个dist目录。
